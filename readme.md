# dark framework

runtime inspired by objective-c, with a java/dotnet inspired framework mashup.

Uses clang with xtensions and c99.


### testing

    cd test
    doran update
    mkdir build
    cd build
    make
    ./unit

### example

a port of the breakout game from learnopengl.com
